# 📝 Django TODO List Application

A clean, modern, and user-friendly TODO list web application built with **Django**. Manage your tasks efficiently by creating, updating, deleting, and toggling completion status. Organize tasks using tags for better categorization.

---

## ✨ Features

- Create, update, and delete tasks  
- Mark tasks as **Complete** or **Undo** completion  
- Assign and manage tags for tasks  
- Responsive and modern UI design  
- Clean and maintainable Django project structure  

---

## 🛠️ Technologies Used

- Python 3.12  
- Django 5.2  
- HTML5 & CSS3  
- (Optional) Bootstrap 5 for layout and styling
